ClovaChat Docs

Step-by-step help for connecting, chatting, moderating, and building self-hosted chat tools.

Getting Started

  1. Open Server Connection in the left sidebar.
  2. Use Twitch Preset for Twitch chat, or enter your own IRC server details.
  3. Enter your nick and password. For Twitch, the password should be an OAuth token that starts with oauth:.
  4. Enter a starting channel such as #mnikevin202.
  5. Click Connect. Server events appear in the Server tab, and joined channels appear in the Channels list in the left sidebar.
  6. The connection card's Connect/Disconnect button always matches your current state, so you don't need to open Server Connection just to reconnect.
  7. Use the Enter channel name field above the Channels list to join a new channel without typing /join.

Onboarding Wizard

  1. The wizard opens automatically the first time ClovaChat starts, or after a first-install reset.
  2. Use Connect Twitch Chat to save your Twitch username and OAuth token.
  3. Use Add Channels to pick channels and decide whether they should be added to Auto Join.
  4. Use Choose Layout to turn stream preview, the user list, timestamps, badges, compact chat, and 7TV emotes on or off.
  5. Use Starter Tools to create optional beginner commands, a timer, and a popup without creating duplicates.
  6. Open Settings and choose Run Onboarding Again any time you want to walk through setup again.

Dashboard

  1. Open Dashboard from the workspace buttons at the top of the window to see joined and Auto Join channels in one view.
  2. Use Sort to prioritize live channels, active chats, alphabetical order, recent activity, or viewer count.
  3. Use filters for All, Live, Offline, Active Chat, Auto Join, and Bot Enabled.
  4. Each card shows live status, viewers when available, category, stream title, users, session messages, activity speed, last message, bot, timers, and logs.
  5. Click a card to open its channel in Chat.
  6. Click the username in the last message preview to open the User Profile Drawer.
  7. Use card actions to open chat, open stream preview, join or leave, add or remove Auto Join, run the first popup action, open logs, or jump to Settings.
  8. Stream status uses the same throttled Twitch polling as live notifications and live tab sorting, so it does not add a separate request loop.

Channel Settings

  1. Open Channel Settings from the active channel status strip, a channel tab right-click menu, Dashboard card actions, Settings, or the Command Palette.
  2. Each setting shows whether it is using the global default or a channel override.
  3. Use Reset to Global Default on one setting, or Reset Channel Settings to clear every override for that channel.
  4. Use Copy Settings to duplicate another channel's setup.
  5. Use Apply To All Channels to copy the current channel's settings to every joined and Auto Join channel.
  6. Per-channel settings can control chat display, bot responses, commands, timers, popups, moderation buttons, logs, notifications, stream preview, favorites, display name, and Auto Join.
  7. If a setting is not saved for a channel, ClovaChat falls back to the global default.

Command Palette

  1. Press Cmd+K on macOS or Ctrl+K on Windows and Linux.
  2. Search for channels, users, settings, commands, timers, popups, stream controls, or app pages.
  3. Use Up and Down to move through results, Enter to run the selected action, and Escape to close.
  4. Recent actions appear before you type so common workflows stay quick.
  5. Actions that would leave a channel, clear a chat view, or remove an Auto Join entry ask for confirmation first.
  6. Unavailable actions stay visible with a short reason, such as needing to connect before joining a channel.

Chat

  1. Select a channel tab.
  2. Type a message in the box at the bottom and press Send.
  3. Use slash commands in the same box, such as /me waves, /join #channel, /part #channel, or custom commands you create.
  4. Type @ and part of a name to use nickname suggestions from the current roster and recent chatters.
  5. Type / and part of a command to use command autocomplete, then press Tab to fill it in.
  6. Use Up and Down in an empty message box to recall messages or commands you sent recently.
  7. Hover over a chat message to mention the user, copy the message, open their profile, or use moderator actions when your account can moderate the channel.
  8. Right-click a username in chat to open the same user actions from a menu.
  9. If you scroll up, ClovaChat keeps your place and shows a New Messages button when fresh chat arrives.
  10. Open Settings, then Chat Display, to change density, font size, timestamps, badges, emotes, mention highlighting, grouped messages, hover actions, and reduced motion.
  11. Messages that mention your nick can trigger desktop notifications when that setting is enabled.
  12. When you're mentioned in a channel you're not currently viewing, its tab shows a small @ badge.
  13. Click the @ button above the channel tabs to see every recent mention across all channels and jump straight to it.

Channels

  1. Join a channel once by typing /join #channel in the message box.
  2. Add a channel to Auto Join from the sidebar, or use Add #channel to Auto Join when viewing a channel that is not already saved.
  3. Drag channel tabs to reorder them. The order is remembered.
  4. Move live channel tabs to the front is enabled by default in Settings.
  5. When enabled, live channels temporarily move before offline channels while Server always stays first.
  6. When a live channel goes offline, it returns to its saved manual tab position.
  7. Right-click a channel tab and choose Leave #channel to part from it.
  8. Use the Server tab for connection messages, notices, and errors.
  9. The Server tab is status-only, so it does not show the roster, popup buttons, auto-join actions, or the message composer.

Streams

  1. Open a channel tab, then click Watch Active Stream.
  2. Use Prev and Next to move between joined or auto-join channels that can be watched.
  3. Use the app controls for quick play, pause, mute, and unmute.
  4. Use the Twitch player controls inside the video for volume, settings, theater-style controls, and fullscreen.
  5. Muted, paused, volume, and playing state are remembered per channel when switching tabs or streams.
  6. By default the player is docked in the sidebar; drag its toolbar out to float it anywhere, and drop it back near the sidebar to re-dock.
  7. Switch to the Twitch Style layout in Settings to put the stream inline in the middle of the chat view with chat on the right, instead of in the sidebar.
  8. The toolbar shows an estimated delay (e.g. "4.2s behind") while a live stream plays.
  9. Click Catch Up to nudge the player closer to the live edge and reduce delay if it's grown large.
  10. If the stream looks frozen (a common symptom after a Twitch ad break), click Reload, or wait — ClovaChat reloads it automatically if it detects no progress for too long.
  11. In Twitch Style layout, if you're not logged in to Twitch, the stream card shows a real Twitch login page instead of the video. Log in there once and it switches to the video automatically; you can also choose "Continue without logging in" to watch anonymously.
  12. Check or change your Twitch login from Settings, including logging out.

Emotes

  1. Twitch emotes show automatically when Twitch sends emote data with chat messages.
  2. 7TV emotes load per channel when 7TV Emotes is enabled in Settings.
  3. Click the emote button next to the message box to browse and insert Twitch or 7TV emotes for the active channel, with search.
  4. If a channel has no 7TV room or 7TV returns an error, normal chat still works.
  5. Turn 7TV Emotes off in Settings if you want plain text for third-party emotes.

Mods & VIPs

  1. Moderator and VIP badges are detected from Twitch IRC tags when users chat.
  2. Known mod and VIP status is remembered per channel so the roster can keep useful role information between messages.
  3. When someone chats again, ClovaChat checks their current tags and updates the remembered role if it changed.
  4. Super moderators and lead moderators are treated as moderator roles in the roster.

Roster Tools

  1. Open a channel and look at the Users list on the right side of Chat.
  2. Click a username to insert that name into the message box for a quick reply.
  3. Right-click a username and choose Monitor to highlight that user's messages in chat.
  4. When you monitor someone, ClovaChat can also save a separate log file of that user's messages if you choose a file.
  5. Right-click a monitored user again and choose Unmonitor to stop highlighting and logging new messages for them.
  6. Right-click a username and choose Hide Chats to mute that user's messages in the current channel view.
  7. Hidden chats are remembered per channel. Choose Unhide Chats from the same right-click menu to show them again.

Popups

  1. Open Popups.
  2. Enter a button label, such as Wave or Discord.
  3. Enter the message or slash command the button should run.
  4. Click Add. The button appears above the message box in Chat.
  5. Use $channel inside a popup command to insert the active channel.

Timed Messages

  1. Open Timers.
  2. Select the channel, interval, and message.
  3. Click Add Timer.
  4. Use the Enabled or Paused switch on each timer to start or stop it.
  5. The Chat topic bar shows Timers Active or Timers Inactive when the current channel has timers.

Commands

  1. Open Commands.
  2. Enter a command name without the slash, such as hello.
  3. Choose mIRC for simple replacements or Python for advanced logic.
  4. For mIRC commands, use $channel, $nick, $*, or numbered args like $1 and $2.
  5. For Python commands, print or send one output line per command or message you want ClovaChat to run.

Bot Builder

  1. Open Bot.
  2. Use Quick Setup to describe what you want, then review the fields it fills in.
  3. Choose whether the rule applies to all channels, this channel, or specific channels.
  4. Choose a trigger: chat command, message contains, exact message, or any message.
  5. Choose Simple reply for templates like Hi {user}, or Advanced Python for custom scripts.
  6. Use {user}, {channel}, {message}, and {args} in simple replies.
  7. Turn a rule off without deleting it by using its On or Off checkbox.

Logs & Backups

  1. Open Settings.
  2. Enable Save chat history between sessions to keep recent chat in the app.
  3. Enable channel logs and choose a folder to save channel log files.
  4. Move live channel tabs to the front is enabled by default and can be turned off under Notifications.
  5. Notify when a channel goes live controls desktop alerts; live tab sorting can work with or without desktop alerts.
  6. Use Backup Settings to export your settings, commands, popups, timers, and bot rules.
  7. Use Import Settings to restore a backup. Reconnect afterward so server settings can apply cleanly.

Reset to First Install

  1. Open Settings and find Danger Zone.
  2. Click Reset ClovaChat to First Install.
  3. ClovaChat creates a backup of your current settings before it resets anything.
  4. Choose whether to also delete local chat history, channel log files, or old backups.
  5. Type RESET to unlock the final reset button.
  6. After reset, ClovaChat clears saved settings, disconnects, and opens the Onboarding Wizard again.

Updates

  1. ClovaChat checks GitHub Releases for newer versions when the app opens.
  2. If a newer version is available, ClovaChat shows a prompt with the new version number.
  3. Choose Download to download the installer for your computer.
  4. On Windows, ClovaChat opens the .exe installer and then closes.
  5. On macOS, ClovaChat opens the downloaded DMG and then closes so you can replace the app.
  6. You can also check manually from Settings by clicking Check for Updates.

Troubleshooting

  1. If messages will not send, confirm you are connected and viewing a channel tab, not Server.
  2. If /join does not open a tab immediately, check Server for Twitch notices or connection errors.
  3. If the user list is incomplete, wait for chat activity or make sure the Twitch token has permission to read chatters.
  4. If 7TV emotes do not load, confirm 7TV Emotes is enabled and check Server for 7TV status lines.
  5. If timed messages continue after changes, pause the timer, reconnect, and verify the timer still exists only once.
  6. If the connection is lost unexpectedly, ClovaChat shows a popup so you can reconnect right away instead of having to find the Connect button.

Known Bugs

Issues we're aware of and how to work around them.

Auto-Updater Not Auto-Detecting Updates

The app's automatic update detection isn't currently notifying you when new versions are available. This is being worked on.

Workarounds:

  • Click the ⟳ Check for Updates button in the top right next to the version number
  • Open Settings and click Check for Updates
  • Restart the app — it automatically checks for updates on startup

Found a Bug?

Help improve ClovaChat by reporting issues! Here's what helps us the most:

  • Screenshots showing the issue
  • Video recordings of the problem occurring
  • Step-by-step reproduction steps so we can recreate it
  • Any relevant details about your setup or what you were doing
